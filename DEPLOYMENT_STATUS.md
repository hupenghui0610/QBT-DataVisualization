### 后端 API
- **端点**: `/api/data/model-daily-sales-trend`
- **功能**:
  - 读取四平台（抖音、小红书、视频号、快手）型号销量数据
  - 读取京东型号销量数据
  - 读取天猫型号销量数据
  - 统一数据格式，按 `日期 + 型号` 聚合
  - 支持缓存机制（48小时）

### 前端图表
- **位置**: 首页 tab 最底部
- **图表标题**: 全渠道型号销量趋势
- **功能**:
  - 多系列折线图，每个型号一条线
  - 指标切换：销量 / 销额
  - 型号显示模式：TOP10 / 全部
  - 日期范围选择器
  - 数据缩放（内置 + 滑块）
  - 图例滚动显示
  - 悬停提示显示详细数据
  - 周期合计统计
